# Web4U Media Ltd website

A colourful Vite + React landing page for Web4U Media Ltd.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Publish on Vercel

1. Create a new GitHub repository.
2. Upload/push these files.
3. In Vercel, choose **New Project** → import the GitHub repo.
4. Vercel should detect Vite automatically.
5. Build command: `npm run build`
6. Output directory: `dist`

## Edit contact email

Change the email link in `src/main.jsx` inside the contact section.
